#Проектная работа "Сложно сосредоточиться"

<h3>Описание проекта</h3>

Выполнено с адаптивной вёрсткой под мобильный экран, планшет. 
Реализовано переключение тем (с учетом настроек браузера): темнаяя, светлая, авто.
